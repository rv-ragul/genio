Web Design: Pixel - Perfect

Just a Boring Description!

Web designing is the process of planning, conceptualizing, and implementing the plan for designing a website in a way that is functional and offers a good user experience. User experience is central to the web designing process. Websites have an array of elements presented in ways that make them easy to navigate.

Web designing essentially involves working on every attribute of the website that people interact with, so that the website is simple and efficient, allows users to quickly find the information they need, and looks visually pleasing. All these factors, when combined, decide how well the website is designed.

General Event Rules

  1. Each participant will have 1 hour to complete the task.
  2. Participants must build a dynamic web application from the given set of themes or topics(On-spot) .
  3. The participants are allowed to choose a framework but it must be from the following list of frameworks:
  4. Bootstrap, Tailwind CSS, Angular, Vannila-framework, Bulma, Materialize CSS

Note:
  1. Team Size: Participants can compete individually or in a team of two, as per their preference.
  2. Equipment: Participants should bring their own laptops or devices with necessary software and development tools pre-installed.
  3. Resources: Participants can use official documentation, tutorials, and online resources specific to the chosen framework.
  4. Judging Criteria: Judging will be based on functionality, user experience, code quality, and creativity in utilizing the chosen framework's features.
  5. Plagiarism is strictly prohibited.
